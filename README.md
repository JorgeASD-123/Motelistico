# Motel Next.js (App Router)

Proyecto mínimo para desplegar en Vercel sin conflictos de ESLint.

## Quickstart
```bash
npm i
npm run dev
```

Si Vercel usa `npm install`, esta versión fija ESLint 8.x para ser compatible con `eslint-config-next@14.2.5`.

## Estructura
```
app/
  layout.tsx
  page.tsx
styles/
  globals.css
public/
```
